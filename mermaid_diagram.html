Mermaid Flowchart for Application

      flowchart TD
          A[Start Application] --> B[Load Model]
          B --> C[Initialize App Class]
          C --> D[Set Default Colors and Brush Size]
          D --> E[Create Canvas and UI Elements]
          E --> F[Bind Paint Event to Canvas]
          F --> G[Main Loop]

          subgraph Paint Functionality
              F --> H[User Draws on Canvas]
              H --> I[Draw Oval on Canvas]
              I --> J[Update ImageDraw Object]
          end

          subgraph Clear Canvas
              E --> K[Clear Button Clicked]
              K --> L[Delete All Canvas Content]
          end

          subgraph Predict Digit
              E --> M[Recognise Button Clicked]
              M --> N[Capture Canvas as Image]
              N --> O[Resize Image to 32x32]
              O --> P[Convert Image to Grayscale]
              P --> Q[Normalize Image Data]
              Q --> R[Predict Using Model]
              R --> S[Display Prediction and Confidence]
          end
